# unix_server_test
유닉스 프로그래밍 과목 과제 수행을 위한 레파지토리
<br>
</br>
Mac Os와 우분투 사이의 gcc 컴파일러 활용시 차이가 있어   
실행환경을 동일하게 설정하기 위해서 AWS 서버에서 실행을 하기 위해 레파지토리를 생성했습니다.
<br>
</br>
과제는 유닉스 운영체제의 ls -R command를 C언어를 기반으로 구현하는 것이고
<br>
</br>
Code파일(lsr.c)   
실행파일(lsr)
<br>
</br> 
2개를 제외한 나머지 파일 및 폴더는 테스트를 위한 더미파일 입니다.
